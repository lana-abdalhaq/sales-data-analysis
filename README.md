# Sales Data Analysis

A data analysis project focused on exploring sales data to extract insights about sales performance, trends, and product demand.

The analysis was performed using Python tools in a Jupyter Notebook environment to clean, analyze, and visualize the data.

---

## Project Overview

This project analyzes a sales dataset to understand business performance and answer key questions related to sales trends, product popularity, and revenue patterns.

The goal is to demonstrate the **data analysis workflow**, including:

- Data cleaning
- Data exploration
- Data visualization
- Extracting business insights

---

## Dataset Information

The dataset contains sales transactions including information about products, sales amounts, and order details.

| Column | Description |
|------|-------------|
| Order ID | Unique identifier for each order |
| Product | Name of the product sold |
| Quantity Ordered | Number of items purchased |
| Price Each | Price per unit |
| Order Date | Date and time of the order |
| Purchase Address | Customer location |

---

## Project Objectives

The analysis answers several important business questions:

| Question | Purpose |
|--------|---------|
| What was the best month for sales? | Identify peak revenue periods |
| Which city had the highest sales? | Understand geographic demand |
| What time should advertisements be displayed? | Optimize marketing timing |
| Which product sold the most? | Identify popular products |
| Which products are frequently bought together? | Discover product combinations |

## Data Analysis Process

The project follows a structured data analysis workflow:

### 1 Data Loading
The dataset is imported into a Pandas DataFrame.

### 2 Data Cleaning
Handling missing values and correcting data types.

### 3 Data Analysis
Aggregating and analyzing sales data to answer key questions.

### 4 Data Visualization
Charts and graphs are created to visualize insights.
